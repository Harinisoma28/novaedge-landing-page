# NovaEdge

NovaEdge is a premium light-theme startup, SaaS, and digital agency landing page built from scratch using only HTML, CSS, and vanilla JavaScript.

## Overview

This project was designed to feel polished, modern, and portfolio-ready. It includes a fixed glassmorphism navbar, cinematic hero section, bento feature grid, animated statistics, premium project showcase, testimonials, contact form validation, and responsive layouts for desktop, tablet, and mobile.

## Features

- Premium responsive landing page design
- Fixed navbar with scroll glass effect
- Animated mobile sidebar menu
- Smooth scrolling and active link highlighting
- Hero section with layered product mockup visuals
- Trusted brands strip
- About and feature highlight sections
- Modern bento feature grid
- Services section with staggered cards
- Animated JavaScript counters
- Projects showcase with hover overlays
- Glassmorphism testimonials
- Gradient CTA banner
- Contact form with vanilla JavaScript validation
- Reveal-on-scroll animations
- Clean semantic HTML and organized CSS

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts: Sora

## Setup Instructions

1. Download or clone the project files.
2. Open `index.html` in any modern web browser.
3. Edit the content, colors, or sections as needed.

No build tools, package managers, or frameworks are required.

## Deployment

You can deploy this static website on:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

For GitHub Pages, upload the files to a repository, then enable Pages from the repository settings.
